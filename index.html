winksplorer

I’m a hobbyist developer, graphic designer, tech tinkerer, and that guy who uses gmake for literally everything.

I don’t follow most modern conventions, which is probably why you're here. I tend to build things that solve my own problems, and my work is mostly in the form of websites, CLI tools, and libraries that prioritize efficiency first and sanity second.

If you're mad about something's design or tech stack, I probably wrote it.

Contact me at wink@winksplorer.net.

Projects

These are links to my GitHub repositories, sorted from most to least useful (in my opinion).

  • summit: A portable & minimal server management web UI for *nix servers, written in Go + vanilla HTML/CSS/JS
  • xappkeys: An app-specific macro keybinding tool for X11 Linux systems with direct device interception
  • kpw7-stats-display: Displays live Linux system stats on a Kindle Paperwhite 7 (or similar)
  • csmarkdoc: Go utility that generates Markdown documentation for C# code
  • snbox: A "Windows Sandbox"-like QEMU frontend written in C
  • minimaterials: Tiny implementation of Material Design for use in lightweight web apps. Mostly vanilla CSS with a little bit of JS
  • feh-apod: A simple Go app that uses feh to set your wallpaper to the current NASA APOD (Astronomy Picture of the Day)
  • markovtext: A horrifically bad "language model" that's really just a Markov chain. Written in Node.js with one dependency, and easily ported to browser
  • fxbf: A Brainfuck interpreter for CASIO fx-9750GIII (or similar) calculators
  • winksplorer.net: This website. Written in vanilla HTML/CSS/JS
